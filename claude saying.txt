  news-overview.js 命令行帮助 (--help)

  用法：node scripts/news-overview.js [选项]

  ┌─────────────────────────────────────────────────────────────────┐
  │  数据源模式 (二选一)                                             │
  ├─────────────────────────────────────────────────────────────────┤
  │  --snapshot        使用快照模式，读取本地 news-overview.json     │
  │                    (默认模式，不调用 Gemini API)                 │
  │                                                                 │
  │  --live            实时模式，抓取 RSS 并调用 Gemini API 生成摘要  │
  │                    (需要有效的 GEMINI_API_KEY)                   │
  └─────────────────────────────────────────────────────────────────┘

  ┌─────────────────────────────────────────────────────────────────┐
  │  输出频道 --channels <value>                                    │
  ├─────────────────────────────────────────────────────────────────┤
  │  预设别名：                                                      │
  │    local     → json + render  (本地文件输出)                    │
  │    publish   → notion + telegram  (发布到外部平台)              │
  │    all       → 全部频道 (notion + telegram + json + render)     │
  │                                                                 │
  │  单独频道 (可用逗号分隔组合)：                                    │
  │    json      → 输出 news-overview.json                          │
  │    render    → 输出 HTML + PNG 图片                             │
  │    notion    → 发布到 Notion 页面                               │
  │    telegram  → 发送到 Telegram 频道                             │
  │                                                                 │
  │  示例：                                                          │
  │    --channels local                                             │
  │    --channels json,render                                       │
  │    --channels notion                                            │
  └─────────────────────────────────────────────────────────────────┘

  ┌─────────────────────────────────────────────────────────────────┐
  │  其他选项                                                        │
  ├─────────────────────────────────────────────────────────────────┤
  │  --zh                翻译输出为中文 (使用 Azure Translator)      │
  │  --snapshotPath <p>  指定快照文件路径 (默认: news-overview.json) │
  └─────────────────────────────────────────────────────────────────┘

  ┌─────────────────────────────────────────────────────────────────┐
  │  完整示例                                                        │
  ├─────────────────────────────────────────────────────────────────┤
  │  # 快照模式，输出到本地文件                                       │
  │  node scripts/news-overview.js --channels local                 │
  │                                                                 │
  │  # 实时抓取，输出 JSON 和图片                                     │
  │  node scripts/news-overview.js --live --channels local          │
  │                                                                 │
  │  # 实时抓取，翻译为中文，输出到本地                                │
  │  node scripts/news-overview.js --live --zh --channels local     │
  │                                                                 │
  │  # 实时抓取，发布到 Notion 和 Telegram                           │
  │  node scripts/news-overview.js --live --channels publish        │
  │                                                                 │
  │  # 实时抓取，全部输出                                             │
  │  node scripts/news-overview.js --live --channels all            │
  │                                                                 │
  │  # 使用自定义快照文件                                             │
  │  node scripts/news-overview.js --snapshotPath ./backup.json     │
  └─────────────────────────────────────────────────────────────────┘

  ┌─────────────────────────────────────────────────────────────────┐
  │  npm 快捷命令                                                    │
  ├─────────────────────────────────────────────────────────────────┤
  │  npm run overview          # 快照模式 + local 频道               │
  │  npm run overview:live     # 实时模式 + local 频道               │
  │  npm run overview:publish  # 快照模式 + publish 频道             │
  │  npm run overview:all      # 快照模式 + all 频道                 │
  └─────────────────────────────────────────────────────────────────┘

  ┌─────────────────────────────────────────────────────────────────┐

  │  NOTION_TOKEN                Notion API Token                   │
  │  TELEGRAM_BOT_TOKEN          Telegram Bot Token                 │
  │  TELEGRAM_CHAT_ID            Telegram Chat ID                   │
  └─────────────────────────────────────────────────────────────────┘
